Sideshow — minimal XHTML5/JS/CSS presentations

  • Sideshow turns webpages into presentations
  • You can use it to write presentations, too
  • You can find/fork the code on github (BSD/MIT/X11 license):
  • http://github.com/wcmaier/sideshow/
  • See the next slide for an example
    • Press spacebar or page down
    • Click the → in the lower right-hand corner
    • Switch to the alternate "sideshow print" stylesheet using your browser to see the slides all at once

Getting started

  • Start with a (X)HTML5 document with <article id="sideshow"> and <section> tags
  • Load sideshow.css and sideshow.js
  • Add a <script> at the end and call sideshow.init("sideshow");
  • The <article> becomes the presentation
    • Each <section> within the <article> will become a slide
  • To see a complete example, take a look at the source for this page

Other stuff

  • The location bar's hash (#slideN) changes as you advance slides
    • You can jump to a slide by editing the hash
    • You can permalink to a slide by including the hash in the URL
  • Theme your presentation with CSS
    • The slides (<section> tags) will be tagged with a .slide class
  • Extend Sideshow
    • There are lots of points to hook in; see sideshow.js
