Sideshow — XHTML5/JS/CSS presentations

  • Sideshow turns webpages into presentations
  • You can use it to write presentations, too
  • You can find/fork the code on github (BSD/MIT/X11 license):
  • http://github.com/wcmaier/sideshow/
  • See the next slide for an example

Getting started

  • Start with a (X)HTML5 document with <article> and <section> tags
  • Load slides.css
  • Load jQuery, jQuery-hashchange and slides.js
  • Add a <script> at the end and call $.slides("article")
  • The <article> becomes the presentation
    • Each <section> within the <article> will become a slide
    • If the <article> has a <header> and/or <footer>, they'll show up on each slide
    • If a <section> has a <header> and/or <footer>, they'll only show up on that slide

Other stuff

  • Theme your presentation with CSS
    • The presentation root will be tagged with a slides class
    • Useful selectors:
      • .slides > header — the presentation header
      • .slides > footer — the presentation footer
      • .slides > section — the slides
      • .slides > section > header — the slide headers
      • .slides > section > footer — the slide footers
  • Extend Sideshow
    • There are lots of points to hook in — see slides.js
